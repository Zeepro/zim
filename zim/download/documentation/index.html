A) HTML Structure - top

The interior pages in this theme are setup with 3 main sections; the header, the main content area, and the footer. The header includes the page title as well as back and home links. The main content section has banner images and the main content of the page (text, images, etc). The footer section contains links which can be customized.

C) Switching Color Schemes (Themes) - top

12 different color schemes are included. In the head of your document you will see links to 2 separate css files. The top file switches the color scheme of your website, the second (style.css) contains universal code which is needed no matter what color scheme you choose. To switch color schemes switch the name of the top css file (from 1.css - 12.css)

D) Customizing Your Theme - top

Each color scheme has 3 different color schemes within it, called data-themes. By default the theme uses the styles setup as "C". To have an item use one of the other 2 data-themes enter either data-theme="b" or data-theme="b" into the element, example:

At the bottom of each color scheme css file (1.css - 12.css) there are custom styles to set the background image, color, and other theme specific styles.

To edit the main colors being used in the theme, edit the colors under the "C" swatch in the css file (1.css - 12.css).

E) Interior Page Header Images - top

The banner images on interior pages are background images on the div with the class="box".

Set another class on the div (about-page in this example), and then link to your background image in the style.css file.

J) Portfolio - top

Galleries used in the portfolio are located on the main portfolio.html file. You link to the gallery by linking to the id of the gallery "page".

The images in the portfolio are included in an unordered list. You can set a title for the image by setting the alt tag.

K) Contact Form - top

To setup your contact form open assets > contact > assets > config.php

Make sure to set your name and the email address for the form to send to (lines 5 & 6).

You can change the text used for items by editing the language.php file in the assets folder.

After submitting the form you are brought to a thank you page. This page is located at the bottom of the contact.php page.

L) Add to Home Bubble - top

The homepage uses a script which creates a "Add to Home" bubble on iPhones and iPads. This reminds people that they can install an icon on their device which will link back to your website. To edit the icon used in the bubble open and edit assets > images > apple-touch-icon.png

To change settings of the bubble open assets > add-bubble > src > add2home.js You can change the animation entering and leaving the page, how long you want the bubble to stay on the page and how long before it shows up again.

M) Creating POPUP Dialogs - top

To create a popup like on the homepage "important link" create a regular page and link to it by adding data-rel="dialog" to the link.

N) Twitter Feed - top

Setup your Twitter feed by opening assets > js > twitter.js. Set your Twitter user name and the amount of tweets to show.
